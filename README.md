# Installation
```bash
# Clone this repository
$ https://github.com/RestuAlamBagaskara/To-Do-List-Web.git

# Go into the repository
# Migrate data
$ php artisan migrate

# Install dependencies
$ composer install
$ npm install

# Run the app
# open terminal #1
$ php artisan serve
# open terminal #2
$ npm run dev
```
